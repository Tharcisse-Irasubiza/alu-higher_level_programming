# python-network_1

This project covers HTTP requests in Python using both the standard
library (`urllib`) and the `requests` library: fetching a URL, reading
response headers, sending POST parameters, handling HTTP errors, working
with JSON APIs, and authenticating against the GitHub API.

## Files

- `0-hbtn_status.py` — fetches a status URL and prints the body (urllib)
- `1-hbtn_header.py` — displays the X-Request-Id response header (urllib)
- `2-post_email.py` — sends a POST request with an email parameter (urllib)
- `3-error_code.py` — prints the body or the HTTP error code (urllib)
- `4-hbtn_status.py` — fetches a status URL and prints the body (requests)
- `5-hbtn_header.py` — displays the X-Request-Id response header (requests)
- `6-post_email.py` — sends a POST request with an email parameter (requests)
- `7-error_code.py` — prints the body or the HTTP error code (requests)
- `8-json_api.py` — searches a user by letter via a JSON API
- `10-my_github.py` — displays a GitHub user id via Basic Authentication
